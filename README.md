Gulp plugin to extract json from jsdoc comments.


## Install

```
npm install gulp-jsdoc-json
```

## Usage 

```javascript
return gulp.src('test/*.js') // get all .js files with jsdoc comments
.pipe(jsdocJson({
	output:'data.json'
}))
.pipe(gulp.dest('./docs')); //output data.json
```
